# Custom Symfony 4 Docker image

This is the image we use at Kinoba to start developing quickly on a Symfony 4 project.

## What's inside?

- phpdockerio/php72-fpm:latest
- php-ext:
    - php7.2-pgsql
    - php-xdebug
    - php7.2-intl
    - php7.2-gd
    - php7.2-phpdbg
- nodejs 8.x
- yarn
- phpunit
